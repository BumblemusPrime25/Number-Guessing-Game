This Number Guessing Game is my third C++ project, built to strengthen my understanding of randomness, user input handling, and program structure. The game generates a random number using proper seeding, then challenges the player to guess it across multiple difficulty levels that adjust the range and number of attempts. I focused on writing more robust logic in this project, including safer input validation, clearer feedback, and a cleaner separation of game functions. It’s a small but meaningful step forward in my C++ learning journey as I move from basic syntax into more thoughtful program design.
